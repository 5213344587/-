# 辅助编程助手
 - 特点
   - 对话辅助编程
   - 用户登录
   - 多轮对话
   - openai接口格式
# 使用方式
 - 1 双击start.bat
 - 2 打开浏览器输入 http://127.0.0.1:8000/
 - 3 用户名和密码：admin/abc
 - 4 关闭两个cmd窗口程序停止
# 使用工具
- 前端 gradio
- 后端服务 llamafile
- 使用模型 deepseek-coder-1.3b-instruct.Q6_K

# 备注
使用本地模型完成功能开发。
受限于模型能力，效果不是很好，如果采用vllm框架的更大参数模型效果会更好。
